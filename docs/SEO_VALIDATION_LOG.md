# SEO manual validation log

Run after meaningful schema or metadata changes.

## Facility profile

1. Deploy or run `npm run dev` and pick any publishable facility URL, e.g.  
   `https://www.starlynncare.com/california/<city>/<facility-slug>`
2. [Google Rich Results Test](https://search.google.com/test/rich-results) — paste the URL; confirm no critical errors on `LocalBusiness` / `BreadcrumbList` / `FAQPage` (if tour questions exist) / `Review` (if reviews exist).

## City / county hub

1. Pick a hub URL, e.g. `https://www.starlynncare.com/california/oakland`
2. [Schema.org validator](https://validator.schema.org/) — paste “View Page Source” JSON-LD blocks or the rendered HTML.
